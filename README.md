#### 나름 있는 기능
* 회원가입, 로그인, 로그아웃, 회원정보 수정
* 글 작성, 수정, 삭제
* 댓글, 검색, 페이징, 체크박스, 이미지 업로드/확인

#### 사용한 기술
* JAVA, Spring
* MS-SQL
* JQuery, Ajax
* 또 뭐 써야하지? 생각나면 써야겠다.
